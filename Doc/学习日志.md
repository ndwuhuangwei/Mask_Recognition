# MobileNet目的

Vgg、Res等的权重文件都太大了，为了在嵌入式设备上部署深度学习模型，发明了MobileNet, 
大大减少了模型参数数量与运算量

MobileNet两个两点
1. Depthwise Convolution(大大减少运算量和参数数量)
2. 增加超参数a, b
    a控制卷积层卷积核个数；b控制输入图像大小
    这两个参数是自定的，不是学习到的
